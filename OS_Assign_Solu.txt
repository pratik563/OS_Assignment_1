##### Problem One #####


a)Navigate and List:

ANS: . cd or cd ~              // To navigate for Home directory.
     . ls                      // To List out the contents in the Home Directory
     . mkdir LinuxAssignment   // To create a new Directory
	 

b)File Management:

Ans: . cd LinuxAssignment  // To move inside the directory.
     . touch file1.txt     // To create a file 
     . cat file1.txt       // To read the content inside the file.
	 
	 
c)Directory Management:

Ans: . mkdir docs          // Created docs directory inside LinuxAssignment directory
     
	 
d)Copy and Move Files

Ans: . cd LinuxAssignment                  // navigated into LinuxAssignment directory
     . cp file1.txt docs/                  // copied the file into docs directory.
     . mv docs/file1.txt docs/file2.txt    // To rename the file 
	 
e)Permissions and Ownership

Ans: . chmod 744 file2.txt             // to set the permission
     . sudo chown $(whoami) file2.txt  // to change owner of current user

	 
f)Final Checklist:

Ans: . cd                              // to go to root directory
     . cd LinuxAssignment              // to go to the LinuxAssignment directory
     . ls                              // List out the content from LinuxAssignment directory
	 
	 
g)File Searching:

Ans: . find . -type f -name "*.txt"    // To find the files with .txt extension

     . grep "India" file1.txt          // To search text from lines of a file.

	 
h)System Information:

Ans: . date                             // To display date,day,time and year.


i)Networking:

Ans: . hostname -I                      // To know the IP address
     . ping google.com                  // To check connectivity
     . Ctrl C                           // To Stop the process

j)File Compression:

Ans: . zip -r docs.zip docs              // To zip the directory
     . sudo apt install zip              // to install zip
     . mkdir extracted_docs              // Created new directory first
     . unzip docs.zip -d extracted_docs  // Unzipped in new directory

	 
k)File Editing:

Ans: . sed -i 's/India/Bharat/g' file1.txt    // To replace old text with new text in a file

e) Ans: . touch input.txt 		// To create input.txt file
        . nano input.txt  		// To edit input.txt file
	. touch output.txt 		// To create output.txt file

	. tr 'a-z' 'A-Z' < input.txt > output.txt    // To convert lowercase letters from input.txt file to uppercase letters and saving in output.txt file.
		
	. cat output.txt 		             // To print the content of file which should be in Uppercase letters.
 
		
f) Ans: . touch duplicate.txt 			// To create duplicate.txt file
	. nano duplicate.txt 			// To edit duplicate.txt

	. sort duplicate.txt | uniq             // To sort the file first then only showcasing unique vales from it.
